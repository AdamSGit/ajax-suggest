Ajax Suggest a jquery plugin by Adam Santoro

Ajax suggest is a small jquery plugin that provide a suggestion list in input elements. Really simple to use, you can (of course) use it with ajax, but also give it directly a javascript map object already existing in your scope.

Github repository Download plugin v0.1.0

Examples

Example 1 : With ajax (countries.json)

(function() {

	$('#example1').ajaxSuggest({ url: '/some-path/controller_example.php' });

})();
<!-- The same directly with html markup -->

<input type="text" name="example1" class="form-control" data-ajax-suggest-input data-url="/some-path/controller_example.php">

Example 2 : With scope object

// Please note that you can't use data-* html markup to initialise the input with scope object.

(function() {

	var days = [
		{ name: 'Monday', memo: 'Oh god, not monday' },
		{ name: 'Tuesday', memo: 'Yesterday was bad' },
		{ name: 'Wednesday', memo: 'Alright, almost the middle of the week' },
		{ name: 'Thursday', memo: 'Yay, friday is tomorrow !' },
		{ name: 'Friday', memo: 'Yaay, tonight is the night' },
		{ name: 'Saturday', memo: 'Ouch, my head' },
		{ name: 'Sunday', memo: 'Oh no, monday is tomorrow' },
	];

	$('#example2').ajaxSuggest({ arr: days, field: 'name' });

})();

How to use

1. How it work

This plugin allow a text input element to display a suggestion list (like google search, for example) which evolve gradually when the user type his request.

This plugin can work in two distinct ways :

  • With ajax

    You give a url to the plugin during initialization. This URL points to a server-side handler (sample php handler in the demo files). This handler must retrieve data from a file or a database, browse the data to filter the related results and return them as JSON object. With this method, the research through the datas must be done by the server-side handler.

  • With scope object

    What if the data you want to search through is in a javascript array of objects, already in your scope ? You can pass this object directly during initialisation as well. (Please note that if you pass both an url and a scope object, the scope object will have the priority). With this method, the research through the datas is done by the plugin directly.

2. Include the files

Before using this plugin, you have to include the files. Copy and adapt the following code :

<!-- Include the css in the <head> tag -->

<link rel="stylesheet" href="assets/css/ajax-suggest-jquery.css">

<!-- And the javascript at the end of the body -->

<script src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script>
<script src="ajax-suggest.jquery.js"></script>

3. Use the plugin

Please refer to examples code to find out the 3 ways to use this plugin

4. Arguments

You can customize the plugin with some arguments described as follow :

  • url : With ajax method, this argument is required and must point to your server-side handler

    Default : null

    $('#myInput').ajaxSuggest({ url: '/my/path/controller_example.php' });

  • arr : When using scope object method, pass it with this argument

    Default : null

    // Please see #example2

  • field : The objects key name used to search through and display results

    Default : 'value'

    $('#myInput').ajaxSuggest({ field: 'name' });

  • delay : In milliseconds. The requests are sent after keystroke when no key was hit the time of this delay

    Default : 400

    $('#myInput').ajaxSuggest({ delay: 1000 });

  • minLength : Minimum input length for the requests to be sent. Before this length, nothing will happend.

    Default : 3

    $('#myInput').ajaxSuggest({ minLength: 3 });

  • noResult : Text displayed in list when no results are matching

    Default : 'No result found'

    $('#myInput').ajaxSuggest({ noResult: 'Oh snap ! Try something else' });
